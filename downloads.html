MFM Original Downloads

If you want to install MFM, download MFM from it's home page by clicking on Download MFM Setup at the left side. If you are interested in the details about how MFM is made, download the complete project inclusive source code by clicking on Download ZIP File or Download TAR Ball at the left side.

To download the user manual go to the GitBook MFM User Manual. If you are interested in general organization stuff go to the GitBook How to Organize Videos on Your Hard Disks.
