MFM Original Downloads

If you want to install lastest version of MFM, cilck on Download Now below. If you are interested in the details about how MFM is made, download the complete project inclusive source code as ZIP File or TAR Ball.

To download the user manual go to the GitBook MFM User Manual, which you can download as ePUB, MOBI or PDF. Some information about the general organization of stuff can be found in the GitBook How to Organize Videos on Your Hard Disks, which you can download as ePUB, MOBI or PDF.

Release Histroy

Each release contains also minor buig fixes and improvements, which are not explicitly mentioned here...

Release 0.2.1

The release fixes the TeraCopy interface interface problem with languages other than English.

Release 0.2.0

This release contains several improvements and bug fixes...

  • Made Tool Tip Regex selectable
  • Made selection based on horizonal resolution
  • Improved Settings
  • Added more Tool Tips
  • Added various minor improvements

Release 0.1.1

This release contains minor fixes like...

  • Improved pictures
  • Updated About tab
  • Updated short instructions
  • Refactored the code

Release 0.1.0

That's the first MFM release to create a stable point on GitHub. I have been using MFM already for over a year, so the implemented functions are already quite stable.
