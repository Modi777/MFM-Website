MFM Original Downloads

If you want to install lastest version of MFM, download the MFM Setup at the left side or above, depending on your screen resolution and zoom. If you are interested in the details about how MFM is made, download the complete project inclusive source code as ZIP File or TAR Ball.

To download the user manual go to the GitBook MFM User Manual. Some information about the general organization of stuff can be found in the GitBook How to Organize Videos on Your Hard Disks.
