Download Movie File Merger for free, without any hidden charges or other tricky stuff...

Application

The latest version of the Movie File Merger is compatible with Windows 7, 8, and 10. Click on Download Now below to get the latest stable version or on Beta to get the latest passing build.

Manuals

The Movie File Merger User Manual explains the general usage of Movie File Merger and how to interact with the supporting programs. You can download it as ePUB, MOBI or PDF. Some information about the general organization of stuff can be found in How to Organize Videos on Your Hard Disks, which you can download as ePUB, MOBI or PDF.



Source

If you are interested in the details about how Movie File Merger is made, download the complete project inclusive source code as ZIP File or TAR Ball. Movie File Merger is open source and the source code can be viewed or downloaded on GitHub.

Of course, you are also very welcome to fork Movie File Merger on GitHub and contribute to the program.


Supporting Apps

Movie File Merger uses or interfaces with a series of other programs. To use the full functionality, you should install the below shown free programs.

  • TeraCopy to copy or move the actual files.
  • Tixati to download the video.
  • WinSCP to download videos over FTP.
