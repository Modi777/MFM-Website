Movie File Merger Bugs

If you find a bug, please post it on the Movie File Merger Bug Reports forum.

GitHub Issue Trackers

If you are on GitHub you can also use one of the below listed issue trackers to submit problems. Movie File Merger is built up out of 4 separate GitHub projects, that's why it has also 4 bug trackers.

  • The Movie File Merger Bug Tracker, where you can log bugs related to the program itself.
  • The Movie File Merger Website Bug Tracker, where you can log bugs related to this web site.
  • The Movie File Merger User Manual Bug Tracker, where you can log bugs related to the Movie File Merger user manual GitBook.
  • The How to Organize Videos on Your Hard Disks Bug Tracker, where you can log bugs related the how to organize videos GitBook.
