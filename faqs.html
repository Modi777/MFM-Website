Frequently Asked Questions

This pages covers frequently asked questions and serves as a training video page for Movie File Merger.

How do I get the files from File Explorer into Movie File Merger?

Just use drag and drop the files or a top level folder.

  • You can drag and drop files from Windows Explorer onto Movie File Merger.
  • One title is listed only once.
  • You can also drag a complete folder.
