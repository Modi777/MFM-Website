Combine

Movie File Merger (MFM) supports you with the selective combination of video collections, like home videos, movies, series, or documentaries on your hard disks. It can be used also for other types of collections, like music, or basically any files, but it is optimized for videos.

View details »

Remember

Movie File Merger remembers all the videos you have in the different categories, like Movies, Documentaries, or Series. So that you'll not copy the same videos again and again. It remembers also all the Garbage movies, which you don;t want to have anymore.

View details »

Maintain

Movie File Merger is designed so that your collection information can be easily updated. It points out bad names for documentaries, movies or series, so that they can be renamed with BileBot. Unrelated files can be moved in the recycle bin and videos can be sucked out of all the subfolders.

View details »

Portable

MMFM is portable, so that you can run it from anywhere, on any hard disk or memory stick. You can even put in in your phone when you use it as memory stick. If Movie File Merger is run in Dropbox it's easier to share lists with other people.

Interfacing

MFM works together with multiple other programs, like FileBot, GOM, VLC, MediaInfo, or TeraCopy. All of them can be downloaded and used for free. Other programs are either directly controlled from MFM or, if they accept file dorps, then then files can bge droped there from the Maintenance interface.
